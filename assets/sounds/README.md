# Музыка для игры

## Добавление музыки для комнат

Поместите музыкальные файлы в эту папку (`assets/sounds/`).

### Поддерживаемые форматы
- `.mp3`
- `.wav`
- `.ogg`

### Именование файлов для комнат

Система автоматически найдёт музыку для каждой комнаты, если файл начинается с названия комнаты. Поддерживаются следующие варианты именования:

#### Для комнаты "Hall" (Холл):
- `hall.mp3`
- `hall_music.mp3`
- `hall-music.mp3`
- `Hall.mp3` (регистр не важен)

#### Для комнаты "Bedroom" (Спальня):
- `bedroom.mp3`
- `bedroom_music.mp3`
- `bedroom-music.mp3`

#### Для комнаты "Kitchen" (Кухня):
- `kitchen.mp3`
- `kitchen_music.mp3`

#### Для комнаты "Shop" (Магазин):
- `shop.mp3`
- `shop_music.mp3`

#### Для комнаты "Playroom" (Игровая комната):
- `playroom.mp3`
- `playroom_music.mp3`

#### Для комнаты "Bathroom" (Ванная):
- `bathroom.mp3`
- `bathroom_music.mp3`

### Общая фоновая музыка

Если для комнаты не найдена специфичная музыка, система попытается найти файлы с "background" или "bg" в названии:
- `background.mp3`
- `bg_music.mp3`

### Музыка для заставки

Для начальной заставки можно добавить музыку:
- `splash.mp3` (предпочтительно)
- `menu.mp3`
- `intro.mp3`

Музыка будет автоматически воспроизводиться при показе заставки и останавливаться при переходе в игру.

### Музыка для мини-игр

Каждая мини-игра может иметь свою индивидуальную музыку. Система автоматически найдёт музыку для каждой игры.

#### Для игры "Memory Game" (Игра на память):
- `memory.mp3`
- `memory_music.mp3`
- `memory-game.mp3`
- `memorygame.mp3`
- `memory_game.mp3`

#### Общая музыка для мини-игр

Если для конкретной игры не найдена специфичная музыка, система попытается найти файлы с "minigame" или "game" в названии:
- `minigame.mp3`
- `game_music.mp3`

### Важно

- Файлы должны быть в формате `.mp3`, `.wav` или `.ogg`
- Регистр букв в названии не важен
- Если для комнаты найдено несколько файлов, будет использован первый найденный
