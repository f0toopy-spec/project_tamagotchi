# Изображения для игры

## Изображения для заставки

Для заставки игры используется **чёрный фон** и логотип по центру.

### Логотип

Поместите логотип игры с одним из следующих имён в эту папку (`assets/images/`):
- `logo.png` (предпочтительно)
- `splash_logo.png`
- `game_logo.png`

**Рекомендации:**
- Формат: PNG (поддерживает прозрачность)
- Логотип будет автоматически отображён **по центру экрана на чёрном фоне**
- Логотип будет иметь лёгкую анимацию пульсации
- Рекомендуемый размер: до 400x400 пикселей
- Если логотип не найден, будет показан текстовый заголовок

### Важно

- Фон заставки всегда **чёрный**
- Если логотип не найден, будет показан текстовый заголовок
- Все изображения должны быть в формате PNG
- Регистр букв в названии файлов не важен (logo.png = Logo.PNG)

## Пример структуры файлов

```
assets/
  images/
    logo.png               (логотип игры)
  sounds/
    splash.mp3            (музыка заставки)
```

## Музыка для заставки

Для заставки можно также добавить музыку в папку `assets/sounds/`:
- `splash.mp3` (предпочтительно)
- `menu.mp3`
- `intro.mp3`

Музыка будет автоматически воспроизводиться при показе заставки и останавливаться при переходе в игру.
